개요

종목 선별 로직

주가 데이터 수신
주가 데이터는 파이썬의 yfinance 라이브러리를 이용하여 매시간 수신한다.
데이, 프리, 애프터 마켓의 데이터는 우선 반영하지 않는다.

데이터 정제
수신한 주가 데이터는 OCHLV이며, 필요에 따라 정규화를 한다.

자동 매매
규칙에 따라서 자동으로 매매 로직에 따라 매수, 홀드, 매도 판단을 한다.
판단을 하면 증권사 API를 이용해서 주문을 넣는다.
